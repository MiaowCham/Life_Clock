# ⏱ 生命时钟

>[!note]
>本项目使用AI生成。  
>This project uses AI generation.

一个网页小项目。计算你的年龄，精确到秒。[点击访问](https://lifeclock.miaowcham.top)  

## 本地运行 Run Locally

**先决条件：** Node.js  
**Prerequisites:**  Node.js

1. 安装依赖项 Install dependencies：
   `npm install`
2. 运行应用程序 Run the app：
   `npm run dev`
